Дневник школьника - мобильное приложение для ведения расписания и успеваемости.
Функционал:
1. Просмотр расписания на неделю
2. Добавление/удаление уроков
3. Просмотр детального расписания дня с домашними заданиями и оценками
4. Расчет среднего балла по предметам
Технологии:
- Архитектура MVVM
- Навигация через Navigation Component
- Room для хранения данных
- Поддержка смены ориентации
- Интернационализация (RU/EN)
